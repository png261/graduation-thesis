# Terraform Infrastructure

This workspace contains infrastructure as code (IaC) for managing cloud resources.

## Getting Started

1. Initialize Terraform:
   ```bash
   terraform init
   ```

2. Review the planned changes:
   ```bash
   terraform plan
   ```

3. Apply the configuration:
   ```bash
   terraform apply
   ```

## Configuration

Update `variables.tf` to customize your infrastructure settings.
